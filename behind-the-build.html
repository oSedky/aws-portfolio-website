Behind the Build

AWS Infrastructure Overview

sedky.net isn’t just a portfolio — it’s a fully deployed AWS project built to demonstrate architectural thinking, production-grade infrastructure, and business alignment.

  • Hosting: Amazon S3 + CloudFront (global CDN)
  • Security: HTTPS via AWS Certificate Manager (ACM)
  • Domain & Email: sedky.net with Microsoft 365, SPF/DKIM/DMARC validated
  • Automation: GitHub + VS Code for version control and updates

This site is part of a broader AWS architecture track, end-to-end projects that demonstrate real-world use of VPCs, Lambda, SES, SageMaker, and more. The goal: deliver AWS-ready cloud solutions with confidence and clarity.
